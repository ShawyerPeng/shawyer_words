# 我的页与设置中心设计

## 目标

把“我的”页升级成真实可操作的设置中心，包含：

- 一级：词典库管理、通用设置、学习设置、数据统计、会员中心、帮助与反馈
- 学习设置：单词书、每日学习计划、学习提醒、默认发音、自动发音、默认显示对话翻译、清除学习进度
- 通用设置：我的语言、外观、主题、字体大小、语言（跳系统设置）
- 数据统计：Heatmap、总词汇量月趋势、每日新词/复习趋势

## 方案

### 1. 设置模块

新增 `features/settings` 模块，提供：

- `AppSettings`
- `AppSettingsRepository`
- `FileSystemAppSettingsRepository`
- `SettingsController`

设置使用本地 JSON 持久化，独立于词典和学习数据。

### 2. 页面结构

`MePage` 只作为入口页，新增三个子页面：

- `GeneralSettingsPage`
- `LearningSettingsPage`
- `StudyStatisticsPage`

会员中心和帮助与反馈本轮保持真实入口和信息页，占位不接远端业务。

### 3. app 级生效项

以下设置要真实影响 app：

- 外观：映射到 `ThemeMode`
- 字体大小：映射到全局文字缩放

由 `ShawyerWordsApp` 持有 `SettingsController` 并驱动根部 `MaterialApp`。

### 4. 学习数据来源

现有学习记录主要落在 `SqliteWordKnowledgeRepository`：

- `isKnown`
- `isFavorite`
- `note`
- `updatedAt`

本轮真实支持：

- 清除学习进度：清空知识记录
- 数据统计：基于知识记录做真实聚合

限制：

- 当前没有完整学习事件流，所以 Heatmap 和每日新词/复习趋势不能做到真正的“逐日事件重建”
- 第一版会基于 `updatedAt` 和知识记录状态做本地聚合，保证数据真实但维度有限

### 5. 学习提醒

本轮支持真实设置项保存：

- App提醒
- 提醒时间

但不接系统通知调度。后续如需通知权限和本地推送，单独扩展。

## 交互

### 我的

一级项顺序：

1. 词典库管理
2. 通用设置
3. 学习设置
4. 数据统计
5. 会员中心
6. 帮助与反馈

### 学习设置

- 单词书：跳现有词书选择能力
- 每日学习计划：选择每日目标数量
- 学习提醒：二级页，包含 App提醒 和 提醒时间
- 默认发音：英式 / 美式
- 自动发音：开关
- 默认显示对话翻译：开关
- 清除学习进度：二次确认

### 通用设置

- 我的语言：应用内语言
- 外观：自动 / 浅色 / 深色
- 主题：主题样式选择
- 字体大小：普通 / 中等 / 加大，含“恢复默认大小”
- 语言：跳系统设置

### 数据统计

顶部文案：

`跟踪每日学习记录，每一点努力都算数`

包含三块图：

- Heatmap：绿色系
- Vocabulary Growth Trend：总词汇量，按月
- Everyday Trend：新词数（红色）、复习词数（绿色）

## 范围

本轮交付：

- 我的页完整入口结构
- 通用设置、学习设置、数据统计页面
- 设置本地持久化
- 外观和字体大小真实生效
- 清除学习进度真实清空
- 数据统计真实聚合展示

不包含：

- 系统通知调度
- 多语言翻译资源体系
- 会员订阅后端
- 帮助反馈提交后端
